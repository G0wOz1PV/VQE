# VQE Optimizer Comparison

Compares optimization methods (Velocity Verlet, COBYLA, L-BFGS-B, SLSQP) for the Variational Quantum Eigensolver (VQE) on the H2 molecule using Qulacs and OpenFermion. Generates convergence plots and a performance table (CSV).

## Dependencies

*   Python 3.7+
*   `qulacs`, `pyscf`, `openfermion`, `openfermionpyscf`
*   `numpy==1.24`, `scipy`, `matplotlib`, `pandas`, `wurlitzer`

## Installation

```
pip install qulacs pyscf openfermion openfermionpyscf wurlitzer numpy==1.24 scipy matplotlib pandas
```
